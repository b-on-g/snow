❄ $bog_snow Demo ❄

Плагин для создания эффекта падающего снега на странице.

Особенности:

  • 40 снежинок с различными параметрами
  • Случайная позиция по горизонтали (0-100%)
  • Случайная скорость падения (5-15 секунд)
  • Случайная задержка старта анимации
  • Случайный размер (10px-30px)
  • Плавная CSS анимация падения
  • Покачивание из стороны в сторону
  • Полупрозрачность для реалистичности
  • Разные символы снежинок (❄ ❅ ❆ •)

Использование:

<div mol_view_root="$bog_snow" mol_view_class="$bog_snow"></div>

Пример в $mol_page:

$my_app $mol_page
	body /
		<= Snow $bog_snow
		<= Content $mol_view
			sub / <= your_content

Снег падает поверх этого контента, но не блокирует взаимодействие с элементами страницы.
